# MoKangMedical OPC 创业者联盟

## 概述

MoKangMedical OPC（Open Pharma Collective）创业者联盟是一个专注于医疗AI的开源生态系统。

## 生态全景

| 领域 | 项目数 | 代表项目 |
|------|--------|----------|
| 药物研发 | 7 | DrugMind, DrugCLIP, AI4S Drug Agent |
| 临床研究 | 7 | PharmaSim, MediChat-RD, Lancet Research |
| AI医疗 | 7 | 天眼, Medical MultiAgent, MediGuard |
| 健康管理 | 8 | 慢康智枢, VoiceHealth, MediSlim |
| 知识平台 | 6 | Digital Sage, 康波, Cloud Memorial |
| 数据科学 | 9 | Virtual Cell, Darwin, EvoX |
| 商业平台 | 7 | OPC Platform, OPC Marketplace |

## 核心项目

- [PharmaSim](https://mokangmedical.github.io/PharmaSim/) - 药品市场仿真平台
- [天眼 Tianyan](https://mokangmedical.github.io/tianyan/) - 商业预测AI
- [MediChat-RD](https://mokangmedical.github.io/medichat-rd/) - 罕见病AI诊疗
- [DrugMind](https://mokangmedical.github.io/DrugMind/) - 药物研发AI
- [慢康智枢](https://mokangmedical.github.io/chronicdiseasemanagement/) - 慢性病管理
- [Digital Sage](https://mokangmedical.github.io/digital-sage/) - 智者课程平台
